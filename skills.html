My Skills

Web Exploitation

Web exploitation involves identifying and leveraging vulnerabilities in web applications. I specialize in securing and testing web applications using various tools and techniques, including:

  • Burp Suite: Proficient in using Burp Suite for web vulnerability scanning, intercepting traffic, and manual testing for issues like SQL Injection, XSS, and more.
  • OWASP ZAP: Skilled in using OWASP ZAP for automating web vulnerability testing and generating reports.
  • SQLmap: Adept at utilizing SQLmap for automating SQL injection discovery and exploitation.
  • DirBuster: Experienced in brute-forcing directories and file locations on a web server to find hidden vulnerabilities.
  • BeEF (Browser Exploitation Framework): Proficient in exploiting browser vulnerabilities using BeEf.

Forensics

Digital forensics is crucial for gathering evidence and analyzing systems after a cyber incident. My forensics skill set includes:

  • Autopsy: Competent in using Autopsy for digital forensics investigations, file recovery, and data analysis.
  • Volatility: Skilled in using Volatility for memory forensics, analyzing RAM dumps to detect malware and process anomalies.
  • Wireshark: Experienced in network packet analysis with Wireshark to identify suspicious network traffic.
  • zsteg: Proficient in using zsteg for steganography detection and analysis in files such as images.
  • Binwalk: Proficient in extracting embedded files from binary images.

Reverse Engineering

Reverse engineering involves deconstructing binaries and software to understand their functionality, identify vulnerabilities, or analyze malware. My reverse engineering skills include:

  • Ghidra: Proficient in Ghidra for reverse engineering and decompiling code to analyze software internals.

Binary Exploitation

Binary exploitation involves identifying vulnerabilities in compiled programs and exploiting them to achieve unintended behavior. My skill set includes:

  • Pwntools: Proficient in using Pwntools for automating binary exploitation tasks in CTFs.
  • gdb & GEF: Skilled in using gdb (GNU Debugger) along with GEF extensions for debugging and analyzing binary crashes.

Programming Languages

As part of my cybersecurity expertise, I am proficient in several programming and scripting languages essential for vulnerability research, exploit development, and automation:

Networking and Telecommunications
