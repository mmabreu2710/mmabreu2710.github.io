CTF Challenges

Welcome to my collection of solved Capture The Flag (CTF) challenges. Each challenge presented below belongs to a different category and tests various aspects of cybersecurity, ranging from web exploitation to reverse engineering. These challenges allowed me to develop and apply skills in real-world cybersecurity scenarios. Explore the categories below to see how I approached and solved these challenges, demonstrating my knowledge and techniques in tackling complex security problems.

WEB EXPLOITATION

In the Web Exploitation challenges, I have tackled vulnerabilities in various web applications. These challenges involved exploiting issues like SQL injection, Cross-Site Scripting (XSS), and broken authentication mechanisms. Through my solutions, I demonstrate techniques to identify and exploit these vulnerabilities to gain unauthorized access or retrieve sensitive data.
